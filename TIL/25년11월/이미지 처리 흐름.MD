### 1. 내재적 크기 지정

```jsx
// 여기에서 height, width 설정 -> 'next.js' 에게 이 파일의 원본 사이징을 알려줌

<Image
    alt="youtube-logo"
    className="h-auto w-[113px]"
    height={24}
    src={`${staticUrl}assets/event/youtube-logo.png`}
    width={122}
/>
```

### 2.1 내재적 크기가 있는 경우의 렌더링 비율 결정 : width 값 지정 & `h-auto` 방식

```jsx
// height, width 로 내재적 크기를 지정한 경우 
// w-[113px] 로 이미지를 결정하면, height 값은 auto 로 하여 알아서 지정이 됨
// so, '원본 비율' 을 유지하면서 렌더링이 됨

<Image
    alt="youtube-logo"
    className="h-auto w-[113px]"
    height={24}
    src={`${staticUrl}assets/event/youtube-logo.png`}
    width={122}
/>

```

### 2.2 내재적 크기가 없는 경우의 렌더링 비율 결정

```jsx
iframe, div 의 경우, '원본 비율(내재적 사이징)' 이 없음 
따라서, aspect-ratio 로 원본 비율을 명시
```
