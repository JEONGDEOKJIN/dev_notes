## 효과 (필요성)

1. '유지보수' 가 빨라짐 
    - '버그' 가 발생하면, '어떤 기능' 이 문제인지 인지하면, '해당 역할을 담당하는 모듈' 을 찾아가면 되기 때문. 

2. '재사용성' 이 높음 
    - 다른 프로젝트에 코드를 이식해서 사용할 수 있음 -> 생산성 증가

3. 'side effect' 가 적음 
    - `격리(isolation)` 가 잘 되어 있음. -> 특정 기능을 고쳤을 때, '다른 기능에 예상치 못 한 오류' 가 발생하는 게 적음
