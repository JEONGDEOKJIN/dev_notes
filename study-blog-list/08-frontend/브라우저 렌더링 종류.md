# CSR, SSR 의 비교

```
HTML 을 어디에서 만드는지 여부

CSR : 브라우저에서 htm 생성
ssr
    : 서버에서 html 생성
    : 네트워크 탭 > 소스 에서 확인 가능
```

# SSG

```
- 빌드시 판단이 됨

- 'blocking 데이터 요구사항' 여부에 따라서 -> blocking 데이터 요구사항이 없으면, SSG 가 됨 ex) 데이터 페칭, 디렉토리에서 [id] 동적으로 요구되는 값들

- blocking 데이터 요구사항이 있으면 -> SSR -> 이거는 브라우저에서 요청이 있을 때 SSR 이 됨
```

### ISR(Increment Static Regeneration)

```
SSG 인데
특정 주기로 fresh data 를 받아서
fresh page 를 생성
```
