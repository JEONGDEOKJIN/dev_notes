


### onClick 에 대한 handler 함수 이름 (#props 로 던질 때에 많이 신경이 쓰임 #props 를 받는 속성 이름도!)
```jsx
// onClick 에 대한 handler 함수 이름 props 로 던질 때 
<FooterButton
    title="파일찾기"
    onClick={() => handleFindFile()}
    bgColor="#2DBAA1"
    textColor="#ffffff"
    buttonWidth="68px"
    buttonHeight="27px"
    titleCSS="!text-[15px] !font-[400]"
/>
```
