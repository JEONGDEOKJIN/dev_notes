# 체크리스트

## 0. 광고 플로우 파악의 3가지 레벨 [P0] (NEW)

단순히 "어떤 버튼을 누르는가"가 아니라 "왜 이렇게 하는가"를 이해해야 함.

### Level 1: 표면적 흐름 (How)

```
기획 → 소재제작 → 세팅 → 집행 → 모니터링 → 보고
```

이건 기본. 하지만 이것만 알면 "시키는 것만 하는 사람"이 됨.

---

### Level 2: 본질적 이해 (Why)

#### A. 광고의 본질 이해

```
[핵심 질문]
- 이 광고의 목적이 뭔가? (인지? 전환? 리타겟팅?)
- 왜 이 매체를 선택했나? (메타 vs 구글 vs 네이버)
- 왜 이 타겟인가?
- 왜 이 소구점인가?
```

- [ ] 고객 구매 여정 (퍼널) 이해
  - 인지 → 관심 → 고려 → 전환
  - 각 단계별로 어떤 광고를 쓰는가?

- [ ] 매체별 특성 이해
  - 메타: 관심사 기반, 시각적 소재 중요
  - 구글 검색: 의도 기반, 키워드가 핵심
  - 네이버: 국내 검색 점유율, 브랜드 검색

- [ ] KGA 교육 상품의 특성 이해
  - 고관여 상품 (비용 높음, 의사결정 기간 김)
  - 구매 여정이 길다 → 리타겟팅 중요

```
[기록]
퍼널별 광고 매체:
- 인지 단계:
- 고려 단계:
- 전환 단계:

매체 선택 이유:
- 메타를 쓰는 이유:
- 구글을 쓰는 이유:
- 네이버를 쓰는 이유:
```

#### B. 현재 회사의 광고 전략 공감하기

```
[핵심 질문]
- 지금까지 어떤 소구점이 효과가 있었나?
- 왜 그 소구점이 먹혔다고 생각하나?
- 타겟 고객은 어떤 사람인가? (페르소나)
- 경쟁사는 어떻게 광고하나?
```

- [ ] 과거 캠페인 히스토리 파악
  - 성공한 캠페인: 왜 성공?
  - 실패한 캠페인: 왜 실패?

- [ ] 현재 소구점 분석
  - "취업률 90%"? "실무 프로젝트"? "국비지원"?
  - 이 소구점을 선택한 근거는?

- [ ] 타겟 고객 이해
  - 나이, 상황, 고민, 니즈
  - 왜 KGA를 선택하나? (vs 경쟁 교육기관)

```
[기록]
타겟 페르소나:
- 연령대:
- 현재 상황:
- 고민/니즈:
- KGA 선택 이유:

현재 소구점:
- 메인 소구점:
- 선택 근거:

과거 성공 캠페인:
- 캠페인명:
- 성공 요인:

과거 실패 캠페인:
- 캠페인명:
- 실패 요인:
```

#### C. 디지털 광고 트렌드 맥락 이해

- [ ] 개인정보 보호 강화 트렌드
  - iOS 14+ 이후 메타 타겟팅 정확도 하락
  - 쿠키리스 시대 대응

- [ ] AI/자동화 트렌드
  - 메타 Advantage+, 구글 Performance Max
  - 자동 타겟팅, 자동 입찰 증가

- [ ] 크리에이티브 중요성 증가
  - 타겟팅 정확도 하락 → 소재가 더 중요해짐
  - UGC(사용자 생성 콘텐츠) 스타일 광고

- [ ] 퍼널 변화
  - 기존: 선형 퍼널 (인지→고려→전환)
  - 현재: 8자형 퍼널 (비선형, 반복 탐색)

```
[기록]
현재 KGA가 적용하고 있는 트렌드:
-

적용하면 좋을 것 같은 트렌드:
-
```

### Level 3: 전략/실행/히스토리 체크리스트

#### 전략 레벨 (Why)

| 항목 | 질문 | 기록 |
|------|------|------|
| 광고 목표 | 현재 캠페인의 최종 목표는? (상담신청? 설명회?) | |
| 타겟 고객 | 이상적인 고객은 어떤 사람? | |
| 핵심 소구점 | 왜 이 메시지가 효과적이라고 보는가? | |
| 경쟁 대응 | 경쟁 교육기관 대비 차별점은? | |
| 매체 선택 이유 | 왜 메타가 효율이 좋다고 판단? | |

#### 실행 레벨 (How)

| 항목 | 질문 | 기록 |
|------|------|------|
| 캠페인 구조 | 캠페인-광고세트-광고 어떻게 구성? | |
| 타겟 설정 | 관심사? 유사타겟? 리타겟팅? | |
| 예산/입찰 | 일 예산? 입찰 전략? | |
| 소재 유형 | 이미지? 영상? 캐러셀? | |
| 성과 지표 | 주로 보는 KPI는? (CTR, CPC, CPA, ROAS) | |

#### 히스토리 레벨 (What worked)

| 항목 | 질문 | 기록 |
|------|------|------|
| 성공 캠페인 | 가장 성과 좋았던 캠페인은? 왜? | |
| 실패 캠페인 | 성과 안 나온 캠페인은? 왜? | |
| 소구점 변화 | 시간에 따라 소구점이 바뀌었나? | |
| 시즌 특성 | 시기별로 성과 차이가 있나? | |
