1순위: Steam 상점 페이지 (조기 오픈, 위시리스트 확보)
2순위: Discord 커뮤니티 (팬 소통, 피드백 수집)
3순위: YouTube 개발자 브이로그 (바이럴 효과 최고)
4순위: Reddit (r/indiegaming, r/gamedev)
5순위: TikTok/Instagram Reels (숏폼 티저)
모바일 게임
1순위: AppLovin/Unity Ads (UA 캠페인)
2순위: Google AdMob (Android 점유율 1위)
3순위: Facebook/Instagram (타겟팅 광고)
4순위: TikTok (게임 바이럴)
언리얼 엔진 게임
• Epic Games Store 출시 시 수수료 12% (Steam 30%)
• 언리얼 마켓플레이스 연동
• Epic MegaGrants 지원금 활용 가능
핵심 인사이트
구분	가장 중요한 퍼스트파티 데이터
PC 인디게임	Steam 위시리스트, 데모 플레이 데이터
모바일 게임	인앱 행동 데이터 (Unity/AppLovin 수집)
커뮤니티	Discord 서버 활동, 유튜브 시청 데이터
결론: 인디게임은 Steam + Discord + YouTube 조합이 가장 효과적이며, 유료 광고보다 데모 출시 + 개발자 브이로그가 위시리스트 확보에 더 효율적입니다.

## 7. KGA 타겟 (게임 + 국비교육) 퍼스트파티 데이터 전략

KGA의 핵심 타겟은 **"게임 개발자가 되고 싶은 국비교육 수강 희망자"**입니다.

### 7-1. 국비교육 플랫폼

| 플랫폼 | 보유 데이터 | 활용 포인트 |
|--------|------------|------------|
| **HRD-Net (고용24)** | 훈련과정 검색 이력, 내일배움카드 발급자 | 과정명 키워드 최적화 (게임, 유니티, 언리얼) |
| **워크넷** | 희망 직종, 구직 이력 | "게임 개발자" 희망자 타겟 |
| **K-디지털 트레이닝** | 부트캠프 수강생, 수료율/취업률 | 신뢰도 마케팅 |

### 7-2. 게임 개발자 취업 커뮤니티

| 플랫폼 | 보유 데이터 | 특징 |
|--------|------------|------|
| **게임잡 (GameJob)** | 이력서, 포트폴리오, 채용공고 열람 | 국내 게임업계 No.1 취업 플랫폼 |
| **인디터웹** | 팀 모집, 프로젝트 참여 | 인디 게임 개발자 커뮤니티 |
| **게임코디** | 기술 Q&A, 게시글 활동 | 게임 프로그래머 커뮤니티 |
| **게임 개발자 Discord** | 서버 활동, 실시간 소통 | 예비 개발자 네트워크 |

### 7-3. 교육/부트캠프 비교 플랫폼

| 플랫폼 | 보유 데이터 | KGA 활용 |
|--------|------------|----------|
| **부트텐트** | 과정 비교, 리뷰 | 게임 개발 과정 등록 |
| **런프리** | 부트캠프 검색 | 국비지원 과정 노출 |
| **인프런** | 강의 시청 이력 | 게임 개발 입문자 타겟 |

### 7-4. KGA 자체 수집 가능 데이터

```
📌 즉시 수집 가능
├─ 상담 신청 폼: 이름, 연락처, 관심 과정
├─ 웹사이트 행동: GA4 이벤트 (페이지뷰, 과정 클릭)
├─ 설명회 참석자: 웨비나/오프라인 등록 DB
└─ 체험 수업 신청자: 전환 가능성 높은 리드

📌 중장기 축적
├─ 수료생 DB: 취업처, 연봉, 후기
├─ 포트폴리오: 졸업 프로젝트 데이터
└─ Alumni 네트워크: 졸업생 커뮤니티
```

### 7-6. 핵심 타겟 페르소나

| 페르소나 | 특성 | 접점 채널 |
|---------|------|----------|
| **취준생 (22-28)** | 대졸, 내일배움카드 보유 | HRD-Net, 게임잡 |
| **전직 희망자 (25-35)** | 타 직종 → 게임 개발 | 네이버 검색, 부트텐트 |
| **게임 덕후 (20-30)** | 플레이 → 제작 희망 | Discord, YouTube |
| **군 전역자 (23-26)** | 진로 탐색 중 | 고용24, 워크넷 |
