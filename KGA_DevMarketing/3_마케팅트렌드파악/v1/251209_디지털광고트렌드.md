# 디지털 광고 트렌드 파악

> 작성일: 2024.12.09
> 목적: 현재 디지털 광고 환경 이해 및 KGA 적용 방안 도출

---

## 요약

```
[2024-2025 핵심 트렌드]
1. 개인정보 보호 강화 → 타겟팅 정확도 하락
2. AI/자동화 캠페인 확대 → 수동 최적화 감소
3. 크리에이티브 중요성 증가 → 소재가 성패 좌우
4. 퍼널 변화 → 비선형 고객 여정 대응 필요

[시사점]
"타겟팅에 의존하던 시대 → 크리에이티브 + AI 자동화 시대"
```

# 1. 개인정보 보호 강화

## 배경

```
[타임라인]
2021.04 - iOS 14.5 ATT(App Tracking Transparency) 적용
2023.07 - 구글 서드파티 쿠키 지원 중단 발표 (2024 → 2025 연기)
2024~   - 쿠키리스 시대 본격화 예고
```

## 변화 내용

### iOS 14+ ATT 영향

| 항목 | Before | After |
|------|--------|-------|
| 메타 픽셀 추적 | 거의 100% 추적 | 옵트인 사용자만 (~30%) |
| 전환 데이터 | 실시간 정확한 데이터 | 지연/모델링 데이터 |
| 리타겟팅 정확도 | 높음 | 크게 하락 |
| 유사 타겟 품질 | 정확함 | 품질 저하 |

### 실제 영향 사례

```
[사례 1: D2C 브랜드 A사]
- iOS 14 이후 메타 ROAS 40% 하락
- 전환 추적 누락으로 최적화 어려움
- 대응: 전환 API(CAPI) 설치 + 크리에이티브 다양화

[사례 2: 교육 업계 B사]
- 리타겟팅 전환율 50% 감소
- 유사 타겟 효율 저하
- 대응: 퍼스트파티 데이터 강화 + 브로드 타겟 테스트

[사례 3: 이커머스 C사]
- 전환 보고 7일 지연
- 실시간 의사결정 어려움
- 대응: GA4 전환 데이터 병행 + 모델링 수용
```

## 대응 전략

### 필수 대응

```
[1] 전환 API (Conversions API) 설치
- 서버 사이드 전환 추적
- 픽셀만 쓰는 것보다 전환 데이터 30~40% 더 수집

[2] 퍼스트파티 데이터 강화
- 이메일, 전화번호 등 자체 수집 데이터 활용
- CRM 연동을 통한 맞춤 타겟 생성

[3] 광역 타겟팅 (Broad Targeting) 테스트
- 세부 타겟팅보다 넓은 타겟 + AI 최적화
- 메타 Advantage+ 활용
```

### KGA 적용 체크리스트

- [ ] 전환 API 설치 여부 확인
- [ ] 퍼스트파티 데이터 (상담신청 DB) 활용 가능 여부
- [ ] 현재 타겟팅 전략 점검 (세부 vs 광역)

# 2. AI/자동화 캠페인

## 배경

```
[흐름]
수동 입찰 → 자동 입찰 → 완전 자동화 캠페인

- 메타: Advantage+ Shopping, Advantage+ App
- 구글: Performance Max (PMax)
- 네이버: 스마트 채널 등
```

## 주요 자동화 캠페인

### 메타 Advantage+

| 기능 | 설명 | 특징 |
|------|------|------|
| Advantage+ Shopping | 이커머스 전용 자동화 | 소재/타겟/배치 자동 |
| Advantage+ Creative | 소재 자동 최적화 | 텍스트, 이미지 변형 |
| Advantage+ Audience | 타겟 자동 확장 | 설정한 타겟 넘어서 확장 |
| Advantage+ Placements | 배치 자동 최적화 | 피드, 스토리, 릴스 등 |

```
[사례: 교육 업계 Advantage+ 활용]

기존 방식:
- 관심사 타겟 3개 세트
- 수동 배치 설정
- 결과: CPA 50,000원

Advantage+ 적용:
- 브로드 타겟 + Advantage+ Audience
- 자동 배치
- 결과: CPA 38,000원 (24% 개선)

[인사이트]
- AI가 학습할 충분한 전환 데이터 필요 (주 50건 이상)
- 소재 다양성이 중요 (AI가 테스트할 소재 필요)
```

### 구글 Performance Max (PMax)

| 항목 | 설명 |
|------|------|
| 개념 | 하나의 캠페인으로 모든 구글 지면 커버 |
| 지면 | 검색, 디스플레이, 유튜브, 지메일, 디스커버, 맵 |
| 특징 | AI가 예산/타겟/소재/배치 자동 최적화 |

```
[사례: 교육 업계 PMax 활용]

Before (개별 캠페인):
- 검색 캠페인 + 디스플레이 캠페인 + 유튜브 캠페인
- 각각 관리 필요
- 총 CPA: 45,000원

After (PMax):
- 하나의 PMax 캠페인
- 에셋 그룹에 다양한 소재 업로드
- CPA: 40,000원 (11% 개선)

[주의점]
- 브랜드 검색어 제외 설정 필요
- 충분한 전환 데이터 필요 (월 30건 이상 권장)
- 블랙박스라 세부 분석 어려움
```

## KGA 적용 체크리스트

- [ ] 메타 Advantage+ 기능 사용 현황 확인
- [ ] 구글 PMax 테스트 여부
- [ ] 주간 전환 데이터 50건 이상인지 확인 (AI 학습용)

# 3. 크리에이티브 중요성 증가

## 배경

```
[변화 흐름]
과거: 타겟팅 정확도가 성과 좌우 (70%)
현재: 크리에이티브가 성과 좌우 (70%)

이유:
1. 타겟팅 정확도 하락 (개인정보 보호)
2. AI가 타겟팅 최적화 → 사람은 소재에 집중
3. 소비자 광고 피로도 증가 → 눈에 띄는 소재 필요
```

## 효과적인 크리에이티브 트렌드

### UGC (User Generated Content) 스타일

```
[정의]
브랜드가 만든 광고가 아닌 것처럼 보이는 콘텐츠
- 실제 사용자 후기 영상
- 인플루언서 리뷰 스타일
- 스마트폰으로 찍은 듯한 자연스러운 영상

[사례: 교육 업계]

기존 광고:
- 깔끔한 디자인의 이미지
- "6개월 완성 게임 개발자"
- CTR: 0.8%

UGC 스타일:
- 수강생 인터뷰 영상 (스마트폰 촬영)
- "3개월 전엔 코딩 1도 몰랐는데..."
- CTR: 2.1% (162% 상승)

[왜 효과적인가?]
- 광고 회의감 감소 ("광고 같지 않아서 봄")
- 공감대 형성 ("나도 저럴 수 있겠다")
- 진정성 전달
```

### 숏폼 콘텐츠 (Reels, Shorts)

```
[트렌드]
- 메타: 릴스 노출 증가, 릴스 광고 효율 상승
- 유튜브: 쇼츠 광고 도입
- 틱톡: 숏폼 광고의 원조

[사례: 교육 업계 숏폼 활용]

릴스 광고 (15초):
- Hook (0-3초): "코딩 배우는 데 1년? 아니요"
- 내용 (3-12초): 수강생 성장 과정 빠르게
- CTA (12-15초): "지금 바로 시작하세요"
- 결과: 일반 피드 광고 대비 CPA 30% 낮음

[핵심 원칙]
- 3초 안에 후킹
- 세로형 (9:16)
- 자막 필수 (무음 시청 대비)
```

### 크리에이티브 다양성

```
[원칙]
하나의 소재로 오래 가는 시대 종료
→ 다양한 소재를 빠르게 테스트하는 시대

[권장 구조]
- 캠페인당 최소 3-5개 소재
- 주 1회 이상 새 소재 추가
- 성과 낮은 소재는 빠르게 교체

[소재 유형 믹스 예시]
1. 이미지 (정적)
2. 영상 (15-30초)
3. 캐러셀 (스토리텔링)
4. UGC 스타일
5. 프로모션/혜택 강조
```

## KGA 적용 체크리스트

- [ ] 현재 소재 유형 파악 (이미지만? 영상도?)
- [ ] UGC 스타일 소재 테스트 여부
- [ ] 릴스/숏폼 광고 사용 여부
- [ ] 소재 교체 주기 확인

# 4. 퍼널 변화 (8자형 퍼널)

## 기존 퍼널 vs 새로운 퍼널

### 기존: 선형 퍼널

```
인지 → 관심 → 고려 → 구매
 ↓      ↓      ↓      ↓
(한 방향으로만 진행)
```

### 현재: 8자형 퍼널 (Messy Middle)

```
        ┌──────────────────┐
        │                  │
   탐색 ←──────────────────→ 평가
        │                  │
        │   반복/순환      │
        │                  │
        └────────┬─────────┘
                 ↓
               구매
```

```
[구글의 "Messy Middle" 연구]

- 소비자는 선형으로 이동하지 않음
- "탐색(Exploration)"과 "평가(Evaluation)"를 반복
- 여러 브랜드를 비교하며 왔다 갔다
- 구매 결정까지 평균 터치포인트 20+회

[교육 업계 특성]
- 고관여 상품 → 더 긴 의사결정 기간
- 여러 교육기관 비교 검색
- 후기, 취업률, 커리큘럼 반복 탐색
```

## 대응 전략

### 멀티 터치포인트 전략

```
[각 단계별 대응]

1. 탐색 단계 (Exploration)
   - 넓은 타겟으로 인지도 확보
   - 콘텐츠: "게임 개발자 되는 법", "국비지원 교육 종류"
   - 매체: 유튜브, 디스플레이, SNS

2. 평가 단계 (Evaluation)
   - 리타겟팅으로 재방문 유도
   - 콘텐츠: 수강 후기, 취업 성공 사례, 커리큘럼 상세
   - 매체: 메타 리타겟팅, 구글 리마케팅

3. 구매 결정 단계
   - 강한 CTA + 혜택
   - 콘텐츠: 마감 임박, 할인, 상담 예약
   - 매체: 검색 광고, 리타겟팅
```

### 리타겟팅 세분화

```
[사이트 방문 깊이별]

1. 홈페이지만 방문
   - 소재: 일반적인 브랜드 소개
   - 빈도: 낮음

2. 코스 페이지 방문
   - 소재: 해당 코스 상세 정보
   - 빈도: 중간

3. 상담 신청 페이지 이탈
   - 소재: 혜택 강조 + 긴급성
   - 빈도: 높음

[시간별]

1. 방문 후 1-3일
   - 적극적 리타겟팅
   - "아직 고민 중이신가요?"

2. 방문 후 7-14일
   - 리마인드
   - "마감이 다가오고 있어요"

3. 방문 후 30일+
   - 재인지
   - 새로운 소구점/혜택
```

## KGA 적용 체크리스트

- [ ] 현재 리타겟팅 세분화 여부
- [ ] 고객 여정별 콘텐츠 차별화 여부
- [ ] 평균 전환까지 터치포인트 파악
